# 럭셔리 인테리어 홈페이지

인테리어 회사를 위한 프리미엄 웹사이트입니다.

## 🎨 특징

- **모던하고 깔끔한 디자인**: 인테리어 회사의 고급스러운 이미지에 맞는 세련된 UI/UX
- **완전 반응형**: 모바일, 태블릿, 데스크톱 모든 기기에서 완벽하게 작동
- **부드러운 애니메이션**: 스크롤 애니메이션과 인터랙션 효과
- **사용자 친화적**: 직관적인 네비게이션과 명확한 정보 구조

## 📋 주요 섹션

1. **홈 (Hero)**: 강렬한 첫인상을 주는 메인 화면
2. **회사소개**: 회사 정보와 통계 데이터
3. **서비스**: 6가지 주요 서비스 소개
4. **포트폴리오**: 완성된 프로젝트 갤러리
5. **문의하기**: 고객 문의 폼과 연락처 정보

## 🚀 시작하기

1. 프로젝트 폴더를 엽니다
2. `index.html` 파일을 웹브라우저에서 실행합니다
3. 별도의 설치나 빌드 과정이 필요없습니다!

## 📁 파일 구조

```
인테리어사이트/
├── index.html      # 메인 HTML 파일
├── styles.css      # 스타일시트
├── script.js       # JavaScript 기능
└── README.md       # 프로젝트 설명
```

## 🎯 커스터마이징

### 색상 변경
`styles.css` 파일의 `:root` 섹션에서 색상 변수를 수정할 수 있습니다:

```css
:root {
    --primary-color: #2c3e50;    /* 메인 색상 */
    --secondary-color: #c9a961;   /* 강조 색상 */
    --text-dark: #333;
    --text-light: #666;
    --bg-light: #f8f9fa;
}
```

### 내용 수정
`index.html` 파일에서 텍스트, 이미지, 연락처 정보 등을 자유롭게 수정할 수 있습니다.

### 이미지 추가
SVG 플레이스홀더를 실제 이미지로 교체하려면:
1. 이미지 파일을 프로젝트 폴더에 추가
2. HTML에서 `<svg>` 태그를 `<img src="이미지경로.jpg">` 로 변경

## 📱 반응형 브레이크포인트

- **모바일**: 480px 이하
- **태블릿**: 968px 이하
- **데스크톱**: 968px 이상

## ⚡ 기능

- **부드러운 스크롤**: 네비게이션 링크 클릭 시 부드럽게 이동
- **모바일 메뉴**: 햄버거 메뉴로 모바일에서 편리하게 사용
- **스크롤 애니메이션**: 요소가 화면에 나타날 때 애니메이션 효과
- **카운터 애니메이션**: 통계 숫자가 카운트업되는 효과
- **문의 폼**: 고객 문의를 받을 수 있는 폼
